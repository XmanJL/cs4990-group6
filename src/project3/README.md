# Project 3 - Multivariate Visualization

## Attribution

We use the dataset from the United States Census Bureau. The dataset to use will be the **Projection of the Population by Single Year of Age, Sex, Race, and Hispanic Origin for the United States: 2016-2060 in a high immigration scenario**.

Here is the [website](https://www.census.gov/data/datasets/2017/demo/popproj/2017-popproj.html) to the dataset.

Check the [CSV file](./high_imm_pop.csv) and [CSV Layout](./CSV_LAYOUT.md) to understand the schema of the dataset.

## Run this Project

1. Run the main program to view and save visualizations into the `/results/project3` folder

```bash
python3 main.py
```
