## Project Structure

The following is how our folders should be organized for this repo.

```
/cs4990-group6
├── src/                # Source code directory
│   ├── project1/       # source code for project 1
│   └── projectN/       # source code for project N
│      └── main.py      # the main program for project N
│      └── jasper.py    # Jasper's work for project N
│
├── results/            # directory to store results (e.g. graphs)
│   ├── project1/       # results for project 1
│   └── projectN/       # results for project N
```

## Set up the project

Install necessary packages in the terminal:

```bash
pip install -r requirements.txt
```

## How do I run code?

First, cd to the directory where your scripts are at:

```bash
cd src/project1
```

Then, run the desired code file:

```bash
python3 main.py
```
