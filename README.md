## Project Structure

The following is how our folders should be organized for this repo. Please read if you are contributing towards this project!

```
/cs4990-group6
├── src/                # Source code directory
│   ├── project1/       # for project 1
│   └── projectN/       # for project N
|   └── lib/            # resuable functions throughout the project
|
├── results/            # directory to store results of the project (e.g. graphs)
│   ├── project1/       # for project 1
│   └── projectN/       # for project N
```

## Set up the project

Install necessary packages in the terminal:

```bash
pip install -r requirements.txt
```

## How do I run code?

First, cd to the directory where your scripts are at:

```bash
cd src/project1
```

Then, run the desired code file:

```bash
python3 main.py
```
